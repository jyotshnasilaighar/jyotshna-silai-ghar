# ✂️ Jyotshna Silai Ghar
Official website for Jyotshna Silai Ghar featuring stitching tutorials, kurti and blouse designs, tailoring services, and YouTube integration for learning sewing in Odia.

Welcome to the official website project of **Jyotshna Silai Ghar** ❤️  
A platform dedicated to stitching tutorials, tailoring services, and design inspiration.

---

## 🌐 About the Project

This website is created to showcase:

- 🧵 Stitching tutorials (Kurti, Blouse, Sleeve designs)
- 👗 Custom tailoring services
- 🎥 YouTube video integration
- 🖼️ Design gallery
- 📞 Contact and inquiry support

The goal is to build a strong online presence and help beginners learn stitching easily in Odia language.

---

## 🚀 Features

- ✅ Modern & responsive design  
- ✅ Beginner-friendly UI  
- ✅ YouTube video integration  
- ✅ Services showcase (Kurti, Blouse, Sleeve)  
- ✅ Gallery of stitching designs
